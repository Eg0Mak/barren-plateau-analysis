# Analysis of Barren Plateaus in Machine Learning Training

## Анализ «бесплодных плато» при проведении анализа обучения моделей машинного обучения (Barren Plateaus)

</div>

---

## О проекте

Проект исследует эффект **barren plateau** («бесплодного плато») в контексте обучения вариационных квантовых классификаторов и классических нейросетей. Основная цель — понять, как архитектура модели и число кубитов влияют на поведение градиентов в процессе обучения, и является ли барьер barren plateau фундаментальным ограничением квантового машинного обучения.

Эксперимент проводится на двух синтетических датасетах: `Circles` и `Moons`. Для каждой конфигурации вычисляются градиентные характеристики: норма, дисперсия и среднее значение градиентов.

---

## Что такое Barren Plateau

**Barren plateau** («бесплодное плато») — эффект экспоненциального затухания градиентов функции потерь при увеличении числа кубитов и глубины квантовой схемы, впервые описанный в 2018 году (McClean et al.).

$$
\mathrm{Var}\left[\frac{\partial L}{\partial \theta}\right] \to 0 \quad \text{при} \quad n_\text{qubits} \to \infty
$$

Ландшафт функции потерь становится практически плоским в большинстве точек пространства параметров. Оптимизатор перестаёт получать полезный сигнал о направлении спуска и фактически блуждает вслепую — даже такой адаптивный алгоритм, как Adam, не способен с этим справиться.

### Механизм возникновения

Случайная инициализация параметров глубокой квантовой схемы эквивалентна случайному унитарному преобразованию. При достаточно большом числе кубитов такое преобразование равномерно «размазывает» квантовое состояние по всему гильбертову пространству — информация о входных данных в измеримых ожидаемых значениях исчезает вместе с градиентом.

### Чем отличается от локального минимума

Barren plateau — не локальный минимум и не седловая точка. Это **глобальное свойство ландшафта**: градиент мал не в одной точке, а везде одновременно. Выбраться из него стандартными методами оптимизации невозможно — требуются архитектурные решения.

### Диагностические метрики

Для диагностики barren plateau в эксперименте вычисляются три характеристики градиента:

* **Gradient Norm** — норма градиента: общая «сила» сигнала обновления весов. Снижение при росте числа кубитов — первый признак эффекта.
* **Gradient Variance** — дисперсия градиента: ключевой индикатор barren plateau. Теоретически должна затухать экспоненциально с ростом числа кубитов.
* **Gradient Mean** — среднее значение градиента: близость к нулю подтверждает плоский ландшафт.

## Датасеты

### Circles — кольцевая симметрия

Два концентрических кольца точек. Граница между классами:

$$
x_1^2 + x_2^2 = r^2
$$

Параметры: `n_samples = 5000`, `noise = 0.3`, `factor = 0.7`, `random_state = 42`. Данные нормализуются через `StandardScaler`.

### Moons — плавная нелинейная граница

Два «полумесяца», расположенных зеркально. Параметры: `n_samples = 5000`, `noise = 0.3`, `random_state = 42`.

---

## Архитектуры моделей

### VQC (Variational Quantum Classifier)

Чистая квантовая модель без классических слоёв. Входные данные кодируются через **AngleEmbedding**:

$$
|\psi_0\rangle = \bigotimes_i R_Y(x_i)|0\rangle
$$

Вариационная схема — **StronglyEntanglingLayers**, 2 слоя. Каждый слой выполняет локальные вращения кубитов по осям X, Y, Z и их перепутывание:

```math
R_i(\theta_i) = R_X(\theta_i^x)\, R_Y(\theta_i^y)\, R_Z(\theta_i^z)
```

```math
U(\Theta) = \prod_{l=1}^{L} \Big(\text{EntangleLayer} \cdot \bigotimes_i R_i(\theta_i^{(l)}) \Big)
```

Измерение снимается только с **первого кубита** (оператор Паули Z), выход через сигмоиду. При `n_qubits = 2` — **12 параметров**. Тестируется при `n_qubits = 2, 4, 8`.

**Susceptibility к barren plateau:** высокая — отсутствие классического слоя означает, что при росте числа кубитов дисперсия градиентов должна экспоненциально затухать.

### VQC_hybrid

Гибридная архитектура: квантовый слой как **feature extractor** + классический линейный слой. Ключевое отличие от VQC — измерение снимается **со всех кубитов**:

$$
\varphi(x) = (\langle Z_0\rangle, \langle Z_1\rangle, \ldots, \langle Z_{n-1}\rangle) \in \mathbb{R}^{n_\text{qubits}}
$$

Классический слой `Linear(n_qubits → 1)` принимает вектор квантовых признаков. Используется **3 слоя** Ansatz. При `n_qubits = 2` — **18 параметров**. Тестируется при `n_qubits = 2, 4, 8`.

**Susceptibility к barren plateau:** умеренная — классический слой стабилизирует градиентный сигнал и теоретически должен сглаживать эффект при росте числа кубитов.

### MLP (Multi-Layer Perceptron)

Классическая нейросеть на PyTorch. Вход (2 нейрона) → скрытые слои с ReLU → выход (сигмоида). Оптимизатор Adam, lr = 0.01, 100 эпох.

Тестируется в трёх конфигурациях скрытых слоёв:
* `[8, 8]` — 2 слоя по 8 нейронов
* `[16, 16]` — 2 слоя по 16 нейронов
* `[32, 32]` — 2 слоя по 32 нейрона

**Susceptibility к barren plateau:** отсутствует — классический градиентный спуск не страдает от затухания градиентов при увеличении ширины сети. Служит базовой линией для сравнения.

---

## Вычисление градиентов

Для каждой модели после обучения вычисляется один прямой и обратный проход, после чего собираются все параметрические градиенты:

```python
def compute_gradients(model, X, y, loss_fn=None):
    # Прямой проход → loss → backward()
    # Сбор градиентов всех параметров
    grads = torch.cat([p.grad.detach().view(-1) for p in model.parameters() if p.grad is not None])
    grad_norm = torch.norm(grads).item()      # Gradient Norm
    grad_var  = torch.var(grads).item()       # Gradient Variance
    return grads.numpy(), grad_norm, grad_var
```

Для квантовых моделей (PennyLane + PyTorch) градиенты вычисляются методом **parameter shift** через `TorchLayer`. Для MLP — стандартным автодифференцированием PyTorch.

## Ожидаемые результаты

### VQC: затухание с ростом числа кубитов

При переходе `n_qubits: 2 → 4 → 8` ожидается экспоненциальное снижение Gradient Variance — прямое проявление barren plateau. Gradient Norm также должна снижаться, а Loss_final — расти (ухудшение сходимости).

### VQC_hybrid: частичная защита

Наличие классического линейного слоя должно стабилизировать градиентный сигнал. Ожидается, что дисперсия градиентов снижается медленнее, чем у чистого VQC, а Loss_final остаётся более стабильным при росте числа кубитов.

### MLP: стабильный градиентный поток

Классическая нейросеть не подвержена barren plateau. При увеличении числа нейронов ([8,8] → [16,16] → [32,32]) Gradient Variance и Gradient Norm должны оставаться стабильными или даже расти — полная противоположность квантовым моделям.

## Выводы

### VQC: barren plateau подтверждён

Чистый VQC демонстрирует классическую картину barren plateau:

* Gradient Norm — от **0.000009** до **0.000036**, что в 100–2700 раз меньше, чем у MLP и VQC_hybrid
* Gradient Variance — в диапазоне **1e-12 — 6e-11**, на 3–6 порядков ниже остальных моделей
* Loss_final застревает на **0.640–0.644** против **0.577–0.581** у VQC_hybrid и MLP — оптимизатор фактически не обучается

На Moons Gradient Variance падает с ростом числа кубитов: с **6.11e-11** (2 кубита) до **2.43e-12** (8 кубитов) — снижение в 25 раз. Это прямое эмпирическое подтверждение теоретической формулы затухания ~1/2^n_qubits.

---

### VQC_hybrid: частичная защита, но не полная

Классический слой создаёт «якорь» для градиентного сигнала:

* Gradient Norm на 2–3 порядка выше чистого VQC: от **0.001358** до **0.009810**
* Loss_final снижается до **0.581** — заметно лучше чистого VQC

Однако с ростом числа кубитов barren plateau проявляется и здесь. На Moons:

* Gradient Norm падает с **0.009810** (2 кубита) до **0.000091** (8 кубитов) — снижение в 108 раз
* Gradient Variance — с **4.81e-06** до **1.03e-10**

Классический слой замедляет затухание, но не останавливает его полностью.

---

### MLP: стабильный градиентный поток

MLP демонстрирует принципиально иное поведение — никакого экспоненциального затухания:

* Gradient Norm растёт с увеличением числа нейронов: **0.002030** ([8,8]) → **0.024633** ([32,32])
* Loss_final стабильно достигает **0.576–0.581** — сопоставимо с VQC_hybrid и значительно лучше чистого VQC
* Gradient Variance не затухает при росте ширины сети — полная противоположность квантовым моделям


---

### Итог

Barren plateau — реальный и измеримый эффект. Чистый VQC фактически не обучается при любом числе кубитов из-за исчезающих градиентов. Гибридная архитектура частично решает проблему за счёт классического слоя, но при 8 кубитах эффект возвращается. MLP полностью свободна от этого ограничения — градиентный поток стабилен при любой ширине сети.

## Влияние числа кубитов на barren plateau

Теоретически дисперсия градиентов затухает как:

$$
\mathrm{Var}\left[\frac{\partial L}{\partial \theta_k}\right] \sim \frac{1}{2^{n_\text{qubits}}}
$$

Это означает, что каждый новый кубит вдвое уменьшает полезный градиентный сигнал. При `n_qubits = 8` дисперсия теоретически в 64 раза меньше, чем при `n_qubits = 2` — оптимизатор фактически блуждает вслепую.

VQC_hybrid частично нейтрализует этот эффект: классический слой `Linear(n_qubits → 1)` получает градиенты напрямую через стандартный backprop, минуя квантовую схему. Это создаёт «якорь» для градиентного сигнала и замедляет затухание.
